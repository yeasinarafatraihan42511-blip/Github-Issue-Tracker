Answer to the question no: 1

var, let, and const are used to declare variables in JavaScript. var is function-scoped and can be redeclared. let is block-scoped and can be reassigned but not redeclared. const is block-scoped and its value cannot be reassigned.

Answer to the question no: 2

The spread operator (...) is used to expand or copy elements of an array or object into another array or object.

Answer to the question no: 3

map() creates a new array by transforming each element. filter() creates a new array with elements that match a condition. forEach() loops through elements but does not return a new array.


Answer to the question no: 4

An arrow function is a shorter syntax for writing functions in JavaScript using the => symbol.

Answer to the question no: 5

Template literals are strings written with backticks (`) that allow embedding variables or expressions using ${}.
